# Cierpinski

A C Sierpinski thingy

## Modes 

| Key | Mode          |
|-----|---------------|
| S   | Scale         |
| L   | Left&Right    |
| T   | Scale Coords  |
| V   | Scale Vertices|
| C   | Color Mode    |
| Q   | Quad Mode     |
| N   | Nodouble      |
| W   | Last+2 Mode   |
| E   | Quad Scale I  |
| R   | Quad Scale II |
| U   | Quad Vertscale|
| P   | Penta Mode    |
| O   | Penta Nodouble|
